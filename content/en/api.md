## API Guide

This document describes the authentication, endpoints, requests, and responses needed to use the sDB API.

## Base URL

The sDB API base URL is:

```text
https://sdb.api.takumi-tokunaga.com
```

## Authentication

`/v1/*` endpoints require an API key. Send it with this header:

```http
X-API-Key: $SDB_API_KEY
Accept: application/json
```

For JSON `POST` requests, also send:

```http
Content-Type: application/json
```

API keys are 68-character strings beginning with `tkp_`. If you do not have an API key, use the portfolio Contact page for the acquisition flow:

- [https://takumi-tokunaga.com/contact/](https://takumi-tokunaga.com/contact/)

## Common Contract

- Encoding is UTF-8.
- Responses are JSON.
- Entity IDs are exposed as `publicId`. These are public IDs generated by sDB and are not official MEXT identifiers.
- For endpoints containing `{publicId}`, use a `publicId` returned by a preceding search or list response.
- List endpoints use `limit` and `offset`.
- `prefectureCode` uses values such as `JP-13`.

Example error:

```json
{
  "error": "API_KEY_INVALID",
  "message": "api key is invalid"
}
```

Status code guide:

- `400`: invalid query parameters or request body
- `401`: missing or invalid API key
- `402`: daily credits are insufficient
- `403`: API key is revoked or access is forbidden
- `404`: requested `publicId` not found
- `500`: internal server error

## Institutions

### `GET /v1/institutions`

Search institutions.

Query parameters:

- `q`: search text
- `type`: `university`, `graduate_school`, `junior_college`, `technical_college`, `technical_college_advanced`, `high_school`, or `vocational_school`
- `prefectureCode`: prefecture code such as `JP-13`
- `limit`: number of items to return
- `offset`: pagination offset

Request:

```bash
curl "https://sdb.api.takumi-tokunaga.com/v1/institutions?q=ritsumeikan&type=university&prefectureCode=JP-26&limit=10" \
  -H "X-API-Key: $SDB_API_KEY" \
  -H "Accept: application/json"
```

Response:

```json
{
  "items": [
    {
      "publicId": "Qh1rOPTq",
      "name": "Ritsumeikan University",
      "displayName": "立命館大学",
      "institutionType": "university",
      "prefectureCode": "JP-26"
    }
  ],
  "total": 1,
  "limit": 10,
  "offset": 0
}
```

### `GET /v1/institutions/{publicId}`

Get one institution by public ID. Use a `publicId` returned by `GET /v1/institutions`.

```bash
curl "https://sdb.api.takumi-tokunaga.com/v1/institutions/Qh1rOPTq" \
  -H "X-API-Key: $SDB_API_KEY" \
  -H "Accept: application/json"
```

### `GET /v1/institutions/{publicId}/faculties`

List faculties or graduate schools for one institution.

```bash
curl "https://sdb.api.takumi-tokunaga.com/v1/institutions/Qh1rOPTq/faculties?limit=20" \
  -H "X-API-Key: $SDB_API_KEY" \
  -H "Accept: application/json"
```

Response:

```json
{
  "items": [
    {
      "publicId": "o4xNXqd4",
      "name": "国際関係学部",
      "displayName": "国際関係学部",
      "academicField": "social_sciences"
    }
  ],
  "total": 16,
  "limit": 20,
  "offset": 0
}
```

## Faculties

### `GET /v1/faculties/{publicId}/departments`

List departments for one faculty. Use a `publicId` returned by `GET /v1/institutions/{publicId}/faculties`.

```bash
curl "https://sdb.api.takumi-tokunaga.com/v1/faculties/o4xNXqd4/departments?limit=20" \
  -H "X-API-Key: $SDB_API_KEY" \
  -H "Accept: application/json"
```

Response:

```json
{
  "items": [
    {
      "publicId": "wcdVaYOl",
      "name": "アメリカン大学・立命館大学国際連携学科",
      "displayName": "アメリカン大学・立命館大学国際連携学科",
      "academicField": "social_sciences",
      "academicTrack": "humanities"
    }
  ],
  "total": 2,
  "limit": 20,
  "offset": 0
}
```

## Suggest

### `GET /v1/suggest`

Return lightweight institution suggestions. This endpoint is intended for autocomplete and does not run the total-count query used by `GET /v1/institutions`.

Query parameters:

- `q`: search text
- `type`: institution type filter
- `prefectureCode`: prefecture filter
- `limit`: number of items to return

```bash
curl "https://sdb.api.takumi-tokunaga.com/v1/suggest?q=tokyo&type=university&prefectureCode=JP-13&limit=10" \
  -H "X-API-Key: $SDB_API_KEY" \
  -H "Accept: application/json"
```

## Selection Events

### `POST /v1/selections`

Record that a user selected an institution, faculty, or department. This optional event can be used to improve search quality or ranking. It is not required for read-only search usage.

Request:

```bash
curl "https://sdb.api.takumi-tokunaga.com/v1/selections" \
  -X POST \
  -H "X-API-Key: $SDB_API_KEY" \
  -H "Content-Type: application/json" \
  -H "Accept: application/json" \
  -d '{"type":"institution","publicId":"Qh1rOPTq"}'
```

Body:

```json
{
  "type": "institution",
  "publicId": "Qh1rOPTq"
}
```

`type` is one of `institution`, `faculty`, or `department`. `publicId` is the ID of the selected entity.
